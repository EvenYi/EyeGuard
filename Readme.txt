System Win10

1.Can not open the webcamera?
Try to change the camera = 0 in config.ini into 1 to switch the webcamera.

2.Eyeblink too sensitive ?
Change the eye_ar_thresh value higher like 0.22 . Range (0.1, 0.3). 
The value is bigger the detection is more sensitive.
